# Job Hunt Website

A modern job portal web application built with **Next.js**, **TypeScript**, **Prisma**, and **Neon** (PostgreSQL). This platform allows users to search for jobs, apply for them, and post jobs if they are employers.

## Features

- User authentication (sign in/sign up)  
- Employers can post jobs  
- Users can browse and search jobs by title, type, and location  
- Users can apply to jobs  
- Dashboard for users to view applied jobs  
- Dashboard for employers to manage posted jobs  
- Responsive design with modern UI  

---

## Tech Stack

- **Frontend:** Next.js, TypeScript, Tailwind CSS  
- **Backend:** Next.js API routes, Prisma ORM  
- **Database:** Neon (PostgreSQL)  
- **Authentication:** NextAuth.js  
- **Utilities:** date-fns, react-hook-form
